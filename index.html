Please read before using this tool

This calculator applies the ISHLT GAP coding system for pulmonary antibody-mediated rejection (AMR), based on the 2026 ISHLT Scientific Statement by Calabrese, Levine, Hachem et al.

This is a coding aid, not a clinical decision tool. It does not replace multidisciplinary assessment and should not be the sole basis for any clinical decision.

Data collection: When you use the Submit button, anonymous data are sent to a research registry. This includes the values you entered (G, A, P and non-core elements) and the resulting code. No patient identifiers are collected or transmitted.

Clicking the button below confirms that you have read this notice.
